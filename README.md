# 🚗 Ford Car Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts the selling price of Ford cars using Machine Learning techniques. The dataset was cleaned, analyzed, and transformed before training a Linear Regression model.

The project demonstrates the complete Machine Learning workflow including:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Preprocessing
- Model Training
- Model Evaluation

---

## 📂 Dataset

Dataset: Ford Car Price Prediction Dataset

Target Variable:

- Price

Input Features include:

- Model
- Year
- Transmission
- Mileage
- Fuel Type
- Tax
- MPG
- Engine Size

---

- Python

## 📊 Data Preprocessing

The dataset was prepared using:

- Missing value handling
- Label Encoding / One-Hot Encoding
- Feature Scaling
- Correlation Analysis
- Feature Selection

---

## 🤖 Machine Learning Model

Model Used:

- Linear Regression

Train-Test Split:

- 80% Training
- 20% Testing

---

| Metric | Score |
|---------|-------|
